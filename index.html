Gacha Tier List

Welcome

In Resonance Solstice, characters lean heavily on one another, so judging any one of them in isolation rarely tells the full story — how good a character is depends a lot on who's standing next to them. That's what makes a single, objective strength ranking so hard to pin down.

Instead, this list breaks every character down across five factors: three performance categories — Starter Game, Calculation, and Boss Maxxing — plus Investment Cost and Awaken Value. That breakdown informs one final verdict, set directly for each character rather than calculated: their Gacha Value, shown here as the Gacha Recommendation Tier List.

It's built with new players, free-to-play players, and low spenders in mind.

Recommended Rules

  1. Prioritize new characters over rerun characters.
  2. Prioritize support characters over DPS carries.
  3. If a character looks like it will get stronger later on, it's fine to wait and pull them then.
  4. These are guidelines, not requirements — pull a character you genuinely like regardless.

If you're F2P or a light spender, weigh Gacha Value more heavily. If you spend more and can afford to invest deeper, weigh Awaken Value more heavily instead.

To be honest, this is still a fairly subjective list. I consulted other top players for input, but there are bound to be gaps — please bear with me, and feel free to point out corrections. That said, since this is meant as a pull-value tier list rather than a pure strength tier list, it should be more broadly useful in practice.

Collab characters are not included in this tier list.

Gacha Recommendation Tier List

Not a "who's strongest" ranking — a "who's worth the gacha" one. Each character is broken down across four factors: how they carry you through Starter Game, Calculation, and Boss Maxxing, plus how much they cost to build (Investment Cost) and how they hold up at low Awakens (Awaken Value). The final Gacha Value verdict below is set directly for each character, informed by that breakdown rather than auto-calculated from it.

What do these mean?

Characters

Starter Game, Calculation, and Boss Maxxing use the S–D scale. Investment Cost uses its own Low/Medium/High scale, and Awaken Value uses a five-step scale from Must to Very Low. The final Gacha Value grade is set directly, not calculated from the others. Click any character to see the reasoning behind their ratings.
